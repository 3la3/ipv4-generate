# ipv4-generate
This tool can generate IP addresses for a proxy.
The output is:
93.219.172.34:38789
233.112.22.69:51495
174.182.107.133:9642
152.30.77.84:20651
153.38.82.2:42776
94.84.241.107:3676
After extracting the IPs, go to checkip.py to verify the IP.

requirements:
pip install argparse
pip install requests
pip install socket
pip install random
